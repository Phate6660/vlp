# spwd

**S**hort **PWD**

This prints a shortened version of the specified directory<br>
(or the current one if `.` or no args are specifed).

Example outputs:
```
$ spwd
/h/v/p/v/spwd
$ spwd .
/h/v/u/v/spwd
$ spwd /home/valley/downloads/git
/h/v/d/git
```

## Requirements

bash version:
- bash
- cut

rust version:
- rust
